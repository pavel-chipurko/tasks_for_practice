- `src/RangeProcessor.php` — универсальный класс, который хранит список порогов (`thresholds`) и соответствующих им обработчиков (`handlers`). Он выполняет бинарный поиск, чтобы определить, какой обработчик применять для конкретного значения.

При инициализации ты передаёшь два массива:
- `thresholds` — отсортированные верхние границы диапазонов (например: `[6,9,40,PHP_INT_MAX]`).
- `handlers` — массив callables в том же порядке (каждый callable принимает число и возвращает значение).

При вызове `process($value)` происходит:
Бинарный поиск `findIndex` — находит первый threshold >= value.
Вызывается соответствующий handler и результат округляется до целого.
