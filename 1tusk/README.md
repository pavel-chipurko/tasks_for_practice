код, который проверяет данные формы регистрации и возвращает подробную информацию о результатах проверки.

файл `src/Validator.php` реализует класс `Validator` с одним публичным методом `validateRegistration(array $data): array`. Метод принимает ассоциативный массив полей (`email`, `password`, `repit_password`, `name`, `phone_number`, `came_from`, `date_birth`) и возвращает структуру вида:

```php
[
  'status' => bool,          // true если ошибок нет
  'message' => string,       // "OK" или "Validation failed"
  'errors' => [ ... ]        // массив всех найденных ошибок, сгруппированный по полям
]

сначала метод берёт каждое поле и применяет к нему набор проверок: для email используется filter_var (чтобы корректно проверить формат), для даты — DateTime::createFromFormat('Y-m-d') и вычисление возраста через DateTime::diff.
