- `src/RangeProcessor.php` — класс, который принимает:
  - `thresholds` — отсортированный массив верхних границ диапазонов,
  - `handlers` — массив callables в том же порядке.
  Метод `process(int $value): int` использует бинарный поиск, чтобы найти индекс нужного диапазона, вызывает handler и возвращает результат, округлённый до целого.
- `tests/RangeProcessorTest.php` — тесты на разные кейсы (low/medium/high/identity) и проверку уникальных результатов в диапазонах.

Как работает
- Конфигурация диапазонов выглядит как `thresholds = [6, 9, 40, PHP_INT_MAX]`. Это значит:
  - значения `<=6` — первый обработчик,
  - `7..9` — второй,
  - `10..40` — третий,
  - `>40` — четвёртый (последний обработчик).
- Бинарный поиск гарантирует, что при большом количестве правил выбор обработчика будет быстрым (O(log m)).
