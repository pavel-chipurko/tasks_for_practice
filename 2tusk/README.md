один раз индексируем данные по полям, затем делаем быстрые хеш-поисковые запросы.


- `src/SearchIndexer.php` — класс `SearchIndexer`, который строит индексы `byPrice` и `byName` при создании.
- `tests/SearchIndexerTest.php` — тесты, которые проверяют поиск по цене, по имени, поведение с дубликатами и пустой вход.


при конструировании `SearchIndexer` происходит один проход по входному массиву (ключ -> `['price'=>..,'name'=>..]`) и строятся два словаря:
- `byPrice[price] = [keys...]`
- `byName[name] = [keys...]`
