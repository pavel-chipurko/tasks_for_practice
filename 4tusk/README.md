Нужно: для каждого продукта, у которого `tradeble = true`, сохранить изображение в файл и вернуть информацию о сохранённых файлах

- `src/ImageProcessor.php` — класс `ImageProcessor::process($input, string $imageDir): array`.
  - `input` может быть JSON-строкой или уже декодированным iterable.
  - функция ищет `call` -> `image_name` и `image.base64`, затем проходит по product-блокам и для тех, у кого `tradeble === "true"` или булево `true`, декодирует base64 и сохраняет файл.
- `tests/ImageProcessorTest.php` — тесты, которые создают временную папку, проверяют успешную запись файла и корректный пропуск нетрейдебл-товаров.



- Base64-строка очищается от префикса `data:image/...;base64,`.
- Контент декодируется `base64_decode`.
- Имя файла формируется как `image_name` + короткий хеш (чтобы не затереть уже существующие файлы при одинаковых входах), сохраняется с расширением `.jpeg`.
- Данные записываются в бинарном режиме (`fopen(..., 'wb')`).
